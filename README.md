# Netflix Content Analysis Dashboard

## Project Overview

This project presents an interactive Netflix Content Analysis Dashboard built using Power BI. The dashboard analyzes Netflix's content library to uncover trends in content growth, ratings, genres, directors, countries, and release patterns. The project demonstrates data cleaning, SQL analysis, DAX calculations, and data visualization techniques.

## Tools & Technologies

* Power BI
* SQL
* Power Query
* DAX
* Data Visualization
* Data Cleaning

## Dataset

The dataset contains information about Netflix titles, including:

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Release Year
* Rating
* Duration
* Genre

## Dashboard Features

### Overview Page

* Total Movies and TV Shows
* Content Distribution by Type
* Top Genres
* Most Common Ratings
* Content Growth Trends

### Content Analysis Page

* Top Directors
* Top Countries by Content Count
* Genre Analysis
* Release Year Trends
* Rating Distribution

## Key Insights

* TV-MA is the most common rating on Netflix.
* Movies make up the majority of Netflix content.
* Netflix content additions increased significantly after 2015.
* Drama and International genres dominate the platform.
* The United States contributes the highest number of titles.
